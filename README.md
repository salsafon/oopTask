Overview

This project is an Object-Oriented Programming implementation of a video game rental store management system.
The system simulates how a game rental store manages its inventory, rentals, pricing, and damaged games while tracking revenue and usage statistics.

The system models a real-world video game rental store where:
Each video game has attributes such as title, genre, age rating, popularity, rental price, and rent count.
Customers can rent games based on availability and preferences.
The store tracks rental statistics, due dates, and revenue generation.
Games become permanently damaged after being rented more than 20 times and are removed from circulation.

The system ensures that customers can efficiently find games while the store maintains a well-managed inventory.
